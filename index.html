Redis Entity Bundle

This bundle attempts to make it easier to store objects in Redis by providing a way of mapping object fields to Redis keys / values.

Installation

The simplest way to install the bundle is to add the following to deps:

[PogotcRedisEntityBundle]
git=git@github.com:pogotc/Redis-Entity-Bundle.git
target=/bundles/Pogotc/RedisEntityBundle
version=origin/master

then run bin/vendors install
